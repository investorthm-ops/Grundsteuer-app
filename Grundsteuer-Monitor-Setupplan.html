⚙️ Installation — Schritt für Schritt auf dem Mac Mini

💡 Reihenfolge wichtig Bitte genau in dieser Reihenfolge vorgehen — jeder Schritt baut auf dem vorherigen auf. Zeitaufwand insgesamt: ~4-6 Stunden, verteilt auf 2-3 Abende.

Phase 1 — Grundwerkzeuge (Abend 1, ~2 Std)

1
Homebrew installieren
Der Paketmanager für alles was folgt. Terminal öffnen und diesen Befehl ausführen:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
✓ Kostenlos5 Min
2
Python 3 installieren
Für das Scraping-Skript. Mit Homebrew ganz einfach:
brew install python3
✓ Kostenlos5 Min
3
Python-Bibliotheken installieren
BeautifulSoup (Scraping), Pandas (Datentransformation), openpyxl (Excel lesen):
pip3 install beautifulsoup4 requests pandas openpyxl gspread
✓ Kostenlos5 Min
4
Ollama installieren (lokale KI)
Ollama bringt Mistral auf deinen Mac Mini — keine Cloud, keine API-Kosten:
brew install ollama ollama pull mistral
✓ Kostenlos~20 Min~4 GB Download
5
n8n installieren & starten
n8n ist dein visueller Workflow-Editor. Via npm installieren:
brew install node npm install -g n8n n8n start
Danach im Browser öffnen: http://localhost:5678
✓ Kostenlos15 Min

Phase 2 — n8n für Dauerbetrieb einrichten (Abend 2, ~2 Std)

🐍 Beispiel Python-Scraper (Grundgerüst)

import requests
import pandas as pd
from bs4 import BeautifulSoup

# Statistikportal Excel laden
url = "https://www.statistikportal.de/..."
df = pd.read_excel(url)

# Daten bereinigen
df.columns = ['Gemeinde','BL','Hbs_A','Hbs_B','GewSt']
df = df.dropna(subset=['Hbs_B'])
df['Stand'] = pd.Timestamp.today()

# In SQLite speichern
import sqlite3
conn = sqlite3.connect('grundsteuer.db')
df.to_sql('hebesaetze', conn, if_exists='append')

# In Google Sheets exportieren
import gspread
gc = gspread.service_account()
sh = gc.open("Grundsteuer-Monitor")
sh.sheet1.update([df.columns.tolist()] + df.values.tolist())
⚡ Wichtiger Tipp für den Start Fang nicht mit allen 11.000 Gemeinden an. Starte mit einem Bundesland (z.B. NRW — hat die besten öffentlichen Daten). Wenn das funktioniert, erweiterst du. Lieber ein perfektes kleines Produkt als ein halbfertiges großes.
